## Getting started

Clone the project

```bash
  git clone https://github.com/slano-ls/slanosite
```

Go to the project directory

```bash
  cd slanosite 
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

Open <http://localhost:3000> with your browser to see the result.
